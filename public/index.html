Image Search

User stories:

  • Image search:
    • I can search for images using a search term.
    • I can add an offset value to skip results.
    • I'll get a list of 10 results (in JSON format), each with a title, description, link, thumbnail link and image link.
  • I can get a list of the most recently submitted search strings.

Instructions for image search

  1. The search domain is https://sargalias-image-search.herokuapp.com/api/imagesearch?q=YOUR_SEARCH_TERMS
  2. Replace YOUR_SEARCH_TERMS with the search terms you want to use.
  3. Optionally, add &offset=X at the end, where X is the number of results you want to skip. For example, adding offset=10 would skip the first 10 results, such as if you were on page 2 of google search results.

Examples:

Click this link to search for funny cats: https://sargalias-image-search.herokuapp.com/api/imagesearch?q=funny cats

Click this link to search for funny cats and skip the first 10 results: https://sargalias-image-search.herokuapp.com/api/imagesearch?q=funny cats&offset=10

Instructions for viewing the latest searches

To view the 10 latest searches, visit: https://sargalias-image-search.herokuapp.com/api/imagesearch/latest
