Request Header Parser Microservice

User story:

I can get the IP address, language and operating system for my browser.

Example usage:

Visit the request header parser page (https://sargalias-header-parser-ms.herokuapp.com/api/whoami).

Example output:

{
 "ipaddress": "127.0.0.1",
 "language": "en-GB",
 "software": "Windows NT 10.0; Win64; x64"
}
